# Отчет по уроку 1: Введение в PyTorch

## Введение

PyTorch — это один из наиболее популярных фреймворков для глубокого обучения с подкреплением, разработанный Meta AI (ранее Facebook AI Research). Он предоставляет гибкую и эффективную платформу для создания и обучения нейронных сетей. Ключевой концепцией PyTorch является тензор — многомерный массив данных, аналогичный массивам NumPy, но с поддержкой вычислений на GPU и автоматического дифференцирования.

Данный урок посвящен знакомству с основами PyTorch, настройке рабочей среды и изучению базовых операций с тензорами.

---

## Что нужно сделать

1. **Познакомиться с PyTorch** — изучить основные концепции и возможности библиотеки.
2. **Создать виртуальное окружение** для изоляции зависимостей проекта.
3. **Установить PyTorch** (CPU- или CUDA-версию в зависимости от доступного оборудования).
4. **Изучить работу с тензорами** — создание, манипуляции и базовые операции над ними.
5. **Изучить примеры кода**, представленные в папке `tensors_basics/` и других файлах урока.

## Ход работы

### 1. Создание виртуального окружения

Для изоляции зависимостей проекта используется стандартный модуль `venv`:

```bash
python -m venv env
```

Активация окружения на Windows:

```bash
.\env\Scripts\activate
```

### 2. Установка PyTorch

Установка выполнена с использованием `requirements.txt`, который содержит ссылку на индексный сервер CUDA 12.1:

```
--extra-index-url https://download.pytorch.org/whl/cu121
torch
torchvision
torchaudio
```

Команда для установки:

```bash
pip install -r requirements.txt
```

Данный подход позволяет зафиксировать версию CUDA и обеспечить воспроизводимость окружения.

### 3. Изучение основ работы с тензорами

На этом этапе изучаются:
- Способы создания тензоров (из списков, нулей, единиц, случайных значений и т.д.).
- Атрибуты тензоров (размерность, тип данных, устройство).
- Арифметические операции над тензорами.
- Индексация и срезы.
- Изменение формы тензоров (reshape, view).
- Преобразование между тензорами и массивами NumPy.

Примеры кода находятся в папке `tensors_basics/`.
